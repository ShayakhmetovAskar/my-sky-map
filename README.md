# My Sky Map

Небольшое приложение, отображающее звёзды, планеты, их фазы и вращение.  
Есть обработка пользовательских изображений с помощью `astrometry.net` и наложение их на небесную сферу.  

## Демо

- Основное приложение: [skymapdev.afsh.space](https://skymapdev.afsh.space/)
- Plate-solver: [skymapdev.afsh.space/solve](https://skymapdev.afsh.space/solve)
